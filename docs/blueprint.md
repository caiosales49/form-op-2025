# **App Name**: Gerador de OP

## Core Features:

- Armazenamento de Dados: Utilizar o armazenamento local para salvar os detalhes bancários do usuário, nome completo e valores frequentemente utilizados para uso futuro.
- Entrada de Formulário OP: Permitir que os usuários insiram dados bancários, nome completo e valores de pagamento para gerar a OP.
- Geração de PDF: Gerar um documento PDF com a ordem de pagamento, seguindo o formato da imagem fornecida.
- Entrada de Intervalo de Datas: Permitir que os usuários especifiquem 'data de início' e 'data de término' para especificar de quais períodos é a 'alimentação'. Por exemplo: Dia 1 ao Dia 7 e Dia 8 ao Dia 15.
- Visualização de PDF: Fornecer uma funcionalidade de Visualização de PDF usando a formatação especificada. Utilizar a ferramenta de IA Generativa para analisar visualizações de PDF em relação a estruturas históricas de OP e sugerir possíveis erros ou otimizações para sua construção.

## Style Guidelines:

- Cor primária: Azul suave (#7EC4CF), inspirado no azul oficial frequentemente visto em documentos governamentais ou bancários, para confiabilidade.
- Cor de fundo: Off-white (#F2F2F2), próximo ao branco, mas mais quente e agradável aos olhos.
- Cor de destaque: Verde claro (#A8E6CE), para sugerir atividade financeira e aprovação, ligeiramente diferente em brilho e saturação.
- Fonte do corpo e título: 'Inter', sans-serif, proporcionando uma experiência de leitura moderna e neutra.
- Manter um layout consistente com a imagem fornecida para familiaridade.
- Ícones mínimos e profissionais para representar ações e campos de informação.
- Transições sutis e animações de carregamento para melhorar a experiência do usuário.
